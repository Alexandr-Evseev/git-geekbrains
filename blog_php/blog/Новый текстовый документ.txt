Для запуска на хостинге, загрузите файл из папки SQL, в mysqladmin. (папку после этого можно удалить.)
Исправьте в файле database.php хост, имя базы, пользователя, и пароль к вашей БД.
Создайте новый файл с админским паролем .htpasswd, и загрузите в папку /admin/.
( Сейчас там файл по умолчанию: логин - admin пароль - admin )
Загрузите файлы на хостинг по ftp.
запустите файл fullpath.php  по url:  https://site.ru/admin/fullpath.php ( Вместо "https://site.ru" вставьте ваш домен. )
Скрипт выдаст реальный полный путь до файла .htpasswd на хостинге.
Этот путь вносите в файл /admin/.htaccess и сохраняете.
И все должно работать. Дальше допиливайте по своему усмотрению.
